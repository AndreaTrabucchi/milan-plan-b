# Milan Plan B

Luxury Milan Plan B near Milano Centrale.

This project is a multilingual GitHub Pages website designed for travelers whose original plans changed and who need a fast, elegant alternative in Milan.

## Main pages
- Trip cancelled? What to do
- Dubai cancelled? Alternatives
- Travel disruption hub
- Travel help checklist
- Language selector

## Goal
Guide visitors from high-intent travel disruption searches to a clear next step:
checking live availability on the official Airbnb listing.
